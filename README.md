# 🎬 AI Movie Genre Classifier

An AI-powered web application that predicts movie genres from plot descriptions using a Transformer-based NLP model. The app features a cinematic user interface with animations and an interactive prediction experience.

## 🚀 Features
- Predicts movie genres from text descriptions
- Built using Transformer-based NLP model
- Multi-label genre prediction
- Cinematic UI with animated transitions
- Streamlit web interface
- GPU support (if available)

## 🧠 Tech Stack
- Python
- PyTorch
- Hugging Face Transformers
- Streamlit
- NumPy
- Joblib

## 🎥 Example Input
"A team of astronauts travel through a wormhole to save humanity."

## 🎯 Example Output
- Science Fiction
- Adventure
- Drama

## ⚙️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
